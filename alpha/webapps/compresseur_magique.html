📖 Comment ça marche ? (cliquer pour ouvrir)
Le codage RLE permet de compresser des images en évitant de répéter la même couleur plusieurs fois.

📝 Règle d'écriture : On indique le nombre de pixels suivi de la lettre de la couleur.
- X X = Blanc (Cross / Blank)
- N N = Noir
- B B = Bleu
- R R = Rouge
- V V = Vert
- J J = Jaune

Exemple : 3X 2N signifie 3 pixels blancs suivis de 2 pixels noirs. Chaque ligne de la grille est séparée par une nouvelle ligne dans le code.
