# Education Numérique Webapps & Ressources

This repository contains two main sections: `webapps/` and `ressources/`.

## Webapps
These are interactive games designed for primary school pupils who are working with the *Décodage* school manuals. You can find more information about the manuals on [https://decodage.edu-vd.ch/](https://decodage.edu-vd.ch/).

The available webapps are:
- **binaire_codage.html (Codage binaire - Entraînement PRO):** An interactive tool to learn and practice binary coding.
- **binaire_message.html (Messages Secrets Binaires):** A fun game for encoding and decoding secret messages using binary.
- **binaire_studio.html (Pixel Binaire Pro — Studio de Codage):** A binary pixel art studio that lets pupils create drawings by coloring pixels and understanding their binary representation.
- **bit_de_parite.html (Bit de parité - Entraînement & Magie):** A game/training tool focusing on the parity bit and its magic.
- **routage_reseau.html (Routage réseau - Temps et UTI):** An interactive activity focusing on network routing.

## Ressources
These are free, ad-free, easy-to-use and simple tools specifically designed for teachers.

The available ressources are:
- **bareme.html (Générateur de Barème):** A tool for generating and exporting grading scales (barèmes).
- **tirage.html (Tirage au Sort):** A simple and effective random drawing tool for classroom use.
